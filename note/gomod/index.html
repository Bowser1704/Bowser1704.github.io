下载和校验 Module downloading and verification

一个 GOPROXY, 一个 GOSUM.

GOPROXY 用来代理。

GOSUM 用来校验，确保开发的时候 fetch 的和运行或者构建的时候 fetch 的是同一个。

major version 的升级

semantic import versioning

为了支持开发更新版本时可以一步步来，不同 major version 的依赖被看作不同的依赖，使用不同的 path.

例如

import (
    "rsc.io/quote"
    quoteV3 "rsc.io/quote/v3"
)
